My Vehicle

How it Works

  • Ensure there is functional WIFI within operating environment.
  • Switch on Vehicle. Cleaning mechanism will switch on as well. The blue LED will start blinking and remains switched on once connected to WIFI
  • Place a light source on the end of pipe/tunnel as vehicle destination
  • Place vehicle in tunnel. It will autonomously tracks light source and travels towards destination.
  • If manual control is desired, you can control remotely using this page.
  • On this page, you can remotely control vehicle and servo motor movement with arrow keys.
  • If location or direction tracking is needed, monitor real time luminous flux values in "My Vehicle Data" section.
  • Retrieve Vehicle and switch off if no longer in need.
  • A demonstration video can be found in the home page
